Rules:
- Keep the output practical and ready to paste into a script editor.
- Prefer short, direct sentences unless the task explicitly asks for a longer style.
- Preserve the user's intent, structure, and tone.
- Do not add explanations before the script.
- If the user asks for sections, keep obvious section breaks in the draft.
